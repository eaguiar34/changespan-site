## Goal
By the end of this tutorial, you’ll have a baseline schedule and an update artifact you can defend.

## Steps
1. Create or import your schedule.
2. Set the baseline.
3. Record changes and generate an update.
4. Export artifacts for review.

## Common mistakes
- Overwriting the baseline.
- Making updates without documenting *why* the change happened.
